# electron-vue

> 测试程序

#### Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:9080
npm run dev

# build electron application for production 只生成vue静态文件不会自动打包，打包程序已经关闭，因为打包会失败
npm run build

# 打包成桌面程序 执行如下命令 
electron-builder 

# lint all JS/Vue component files in `src/`
npm run lint

```
